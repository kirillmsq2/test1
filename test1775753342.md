@startuml
title Платёж по QR‑коду через сервис ERIP

'--- Участники -------------------------------------------------
actor "Плательщик" as Customer
participant "Мобильное приложение банка" as MobileApp
boundary "Платёжный сервис ERIP" as ERIP
participant "Anti‑Fraud" as AntiFraud
boundary "Система межбанковских платежей (СМП)" as SMP
participant "POS‑терминал (касса)" as POS

'--- Основной сценарий -----------------------------------------
== Сканирование QR‑кода и запрос оплаты ==
Customer -> MobileApp : Сканировать QR‑код\n(данные корзины)
MobileApp ->> ERIP : CreateEnvoy(корзина) \n(запрос оплаты)
note right of ERIP : Envoy сохраняется в директории RTP

== Проверка риска ==
ERIP -> AntiFraud : GetRiskIndicator(EnvoyId)
AntiFraud --> ERIP : risk‑indicator.ts

== Денежный перевод ==
ERIP -> SMP : TransferFunds(EnvoyId, amount, beneficiary)
SMP --> ERIP : TransferResult(success/failure)

== Обновление статуса и уведомление ==
ERIP --> MobileApp : EnvoyStatus(«оплачен», risk)
MobileApp -> POS : NotifyPaymentSuccess(EnvoyId)

'--- Альтернативный сценарий: регистрация клиента/терминала --
alt Регистрация клиента в ERIP
    Customer -> MobileApp : Запрос регистрации
    MobileApp -> ERIP : RegisterClient(clientData)
    ERIP --> MobileApp : RegistrationResult
end

alt Регистрация терминала в ERIP
    POS -> MobileApp : Запрос регистрации терминала
    MobileApp -> ERIP : RegisterTerminal(terminalData)
    ERIP --> MobileApp : RegistrationResult
end

@enduml
